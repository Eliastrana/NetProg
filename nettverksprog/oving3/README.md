# How to run

## Compile

### Server.cpp
    g++ -o server Server.cpp -pthread -std=c++11

### Client.cpp
    g++ -o client Client.cpp

### WebServer.cpp
    g++ -o webserver WebServer.cpp -pthread -std=c++11


## Run

### Server.cpp

    ./server

### Client.cpp

    ./client

### WebServer.cpp
    ./webserver
